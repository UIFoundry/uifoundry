## Props

The Header Menu Items accepts the following props from PayloadCMS:

<TypeTable
  type={{
    alignment: {
      type: "select",
      description:
        "Alignment position within the header layout (left, center, right)",
      default: "center",
    },
    menuItems: {
      type: "array",
      description: "Array of navigation menu items",
      required: true,
      default:
        "[{label: 'Home', href: '/'}, {label: 'Products', href: '/products'}, ...]",
      fields: {
        label: {
          type: "text",
          description: "Display text for the menu item",
          required: true,
        },
        href: {
          type: "text",
          description:
            "Target URL or anchor link (e.g., '/features' or '#features')",
          required: true,
        },
      },
    },
  }}
/>

## Features

### Responsive Navigation

- **Desktop Display**: Horizontal menu with gap spacing
- **Mobile Display**: Vertical menu for mobile overlay
- **Conditional Rendering**: Shows/hides based on `isMobile` prop
- **Flexible Layouts**: Adapts to different screen sizes

### Menu Item Configuration

- **Dynamic Items**: Configurable array of navigation links
- **Custom Labels**: User-defined text for each menu item
- **Flexible Hrefs**: Support for both routes (`/about`) and anchors (`#contact`)
- **Default Menu**: Includes Home, Products, Services, Contact links

### Styling and Interactions

- **Hover Effects**: Color transitions on menu item hover
- **Typography**: Consistent text sizing and spacing
- **Color Scheme**: Uses theme-aware muted foreground colors
- **Smooth Transitions**: 300ms duration animations

### Responsive Behavior

#### Desktop Mode (`isMobile={false}`)

- Hidden by default on mobile (`hidden lg:block`)
- Horizontal flex layout with 8px gaps
- Small text size for clean appearance

#### Mobile Mode (`isMobile={true}`)

- Visible on all screen sizes (`block`)
- Vertical flex layout with 4px gaps
- Center-aligned text for mobile overlay

## Usage in Custom Header

This component integrates with the Custom Header's layout system:

### Standard Navigation (Desktop)

```tsx
<RenderBlocks
  blocks={(items ?? []).filter((i) => i.alignment === "center")}
  blockComponents={blockComponents}
/>
```

### Mobile Menu (Mobile Overlay)

```tsx
<RenderBlocks
  isMobile={true}
  blocks={(items ?? []).filter(
    (i) => i.blockType === BLOCK_SLUG_HEADER_MENU_ITEMS,
  )}
  blockComponents={blockComponents}
/>
```

## Default Menu Configuration

  { "label": "Products", "href": "/products" },
  { "label": "Services", "href": "/services" },
  { "label": "Contact", "href": "/contact" }
