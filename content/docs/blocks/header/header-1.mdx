## Props

The Header_1 block accepts the following props from PayloadCMS:

<TypeTable
  type={{
    menuItems: {
      type: "Array<{label: string, href: string}>",
      required: true,
      description: (
        <div>
          <p className="pl-4">
            An Array of <b>Nav Items:</b>
          </p>
          <TypeTable
            type={{
              label: { type: "string", description: "Label", required: true },
              href: {
                type: "string",
                description: "Target Url",
                required: true,
              },
            }}
          />
        </div>
      ),
    },
  }}
/>

## Installation

```bash
npx shadcn@latest add @uifoundry/header-1
```
